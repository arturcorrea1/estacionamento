# Estacionamento

## Este projeto é uma simulação de um estacionamento, contendo diversos campos preenchíveis, sendo eles:

### Placa
### Dono (Nome do dono do carro)
### Cpf (Placa do Dono)
### Tipo (Padrão, gestante, deficiente)
### Vagas (A1 - C3)
